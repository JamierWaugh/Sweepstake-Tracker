# Sweepstake tracker
## General information
This sweepstake tracker was created to help practice react and typescript development.

When a ``sweepstake.json`` is populated in ``src/data/`` using the format of ``example_sweepstake.json``, a sweepstake webapp is created that allows users to track the world cup groups, upcoming games, live scores, all while knowing who's team is playing who's.

This sweepstake tracker uses an open source api for the world cup by rezarahiminia on github.
